# 📁 샘플 데이터 사용 가이드

이 폴더의 CSV 파일을 앱의 **① 데이터** 탭에서 업로드하여 시연할 수 있습니다.

## 시나리오 시리즈 (스토리텔링용)

### 📄 `01_bearing_march.csv` — 3월 안정 공정
- **제품**: 베어링 외경 측정 (mm)
- **데이터**: 30 lots × 5 obs = 150 행
- **컬럼**: `Lot`, `Diameter`
- **권장 설정**: LSL=**19.9**, USL=**20.1**, target=**20.0**
- **기대 결과**: Cpk ≈ **1.47** (충분), PPM ≈ 52, Nelson 위반 거의 없음
- **시연 메시지**: "안정된 공정의 모범 사례"

### 📄 `02_bearing_april.csv` — 4월 문제 발생 공정
- **제품**: 동일 베어링 라인, 다음 달 측정
- **데이터**: 30 lots × 5 obs = 150 행
- **컬럼**: `Lot`, `Diameter`
- **권장 설정**: LSL=**19.9**, USL=**20.1**, target=**20.0** (1번과 동일)
- **기대 결과**: Cpk ≈ **0.46** (매우 부족), PPM ≈ 107,893
  - 공구 마모로 인한 점진적 평균 시프트 시뮬레이션
  - Lot 22에 의도적 이상치 추가
- **시연 메시지**: "데이터 교체 → 즉시 진단 변경. SPC로 이상치 검출 → 재작성"

### 📄 `03_surface_roughness.csv` — 비정규 데이터
- **제품**: 가공 부품 표면 거칠기 Ra (μm)
- **데이터**: 25 batches × 6 obs = 150 행
- **컬럼**: `Batch`, `Roughness_Ra`
- **권장 설정**: LSL=**0.5**, USL=**3.5**, target=**1.5**
- **기대 결과**: 정규성 불만족 → Box-Cox λ ≈ -0.11 → 변환 후 분석
- **시연 메시지**: "비정규 데이터도 Box-Cox 한 번 클릭으로 처리"

## 시연 흐름 (3분 발표 권장)

```
① 01_bearing_march.csv  업로드 → Cpk 1.47 확인 (안정)
                ↓
② 02_bearing_april.csv  업로드 → Cpk 0.46 진단 (문제)
                ↓
③ SPC 관리도 → Lot 22 이상치 검출 → 이상치 제거 시뮬
                ↓
④ Compare 탭 → 3월 vs 4월 비교
                ↓
⑤ What-if 탭 → "σ 절반 줄이면 Cpk가?" 시뮬
                ↓
⑥ 03_surface_roughness.csv 업로드 → Box-Cox 변환 시연
                ↓
⑦ LIVE 모드 → 실시간 모니터링 데모
```

자세한 발표 멘트는 `PRESENTATION_SCRIPT.md`를 참고하세요.

## 직접 만들고 싶다면

데이터 페이지의 **"🎲 시뮬레이션 생성"** 영역에서:
- 목표값, 허용오차, 부분군 수/크기, 평균 시프트 등을 조정하여
- 원하는 시나리오의 데이터를 즉시 생성할 수 있습니다.
